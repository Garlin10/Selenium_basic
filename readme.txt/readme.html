Python Selenium Automation Script

This Python script, created by Gyula GÃ¡l, uses Selenium to automate web browser interactions. It provides a set of functions to control a web browser, such as opening URLs, clicking buttons, and entering text into fields.

Features

  • Automatic ChromeDriver Installation: Automatically installs the ChromeDriver required for Selenium.
  • Open a URL: Navigate to a specified URL.
  • Element Visibility Check: Check if a certain element is visible on the page.
  • Wait for Element Visibility: Wait until a specific element becomes visible.
  • Click Button: Click a button on the page.
  • Write Text in Field: Enter text into a text field.
  • Get Current URL: Retrieve the current URL of the browser.

Requirements

  • Python 3.x
  • Selenium
  • chromedriver_autoinstaller

Installation

Before running the script, make sure you have Python installed on your system. You can download and install Python from python.org.

To install the required Python packages, use the following command in your terminal:

pip install -r "requirements.txt"

This will install all the dependencies listed in the 'requirements.txt' file.

Usage

  1. Start the Browser: Call start_browser() to start a new Selenium WebDriver instance.
  2. Open a URL: Use open_url(driver, url) to navigate to the desired webpage.
  3. Interact with the Webpage: Use functions like click_button, write_text_in_field, and is_element_visible to interact with web elements.

Example


driver = start_browser()
open_url(driver, "https://www.example.com")
click_button(driver, '/html/body/div/p[2]/a', By.XPATH)
# ... other actions ...
driver.quit()

Replace 'https://www.example.com' with the URL you want to open and '/html/body/div/p[2]/a' with the XPath or JS path of the element you want to interact with.
